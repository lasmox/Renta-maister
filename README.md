# Rent-a-Meister (RaM)

Projekt za upravljanje i prezentaciju Rent-a-Meister (RaM) sustava.

## Struktura Projekta

Prema planu, projekt se sastoji od sljedećih cjelina:
*   **RaM-website** - Korisničko sučelje i prezentacijska stranica (React + Tailwind)
*   **RaM-webapp** - Pozadinska aplikacija (Python)
*   **RaM-database** - Baza podataka
*   **RaM-documentation** - Dokumentacija projekta
